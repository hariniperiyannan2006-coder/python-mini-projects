# Password Strength Checker (Python)

## Description
This project checks the strength of a password based on basic security rules such as:
- Minimum length
- Use of uppercase letters
- Use of lowercase letters
- Use of numbers
- Use of special characters

It helps users understand whether their password is Weak, Medium, or Strong.

## Technologies Used
- Python 3

## How to Run
1. Download the file `password_strength_checker.py`
2. Open terminal / command prompt
3. Run:
   python password_strength_checker.py

## Learning Outcome
- String handling in Python
- Conditional statements
- Loops
- User input validation
